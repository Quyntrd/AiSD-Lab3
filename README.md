# Сортировки
## Сортировка пузырьком, сортировка Шелла, сортировка расческой
Реализовать алгоритмы сортировки
Посчитать кол-во сравнений и число копирований объектов для массивов.
а) в среднем (сгенерировать 100 случайных массивов)
б) полностью отсортированного массива
в) для обратно отсортированного массива
Построить графики, сравнить эксперементальные результаты с теорией
Доп.задания
Изменринить функции так, чтобы они принимали в себя итераторы, указывающий на начало и конец сортируемого диапозона (подобно <algorithm>)
